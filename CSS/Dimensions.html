Default width of block level elements is 100% of their parent. we can change this behavior by giving custom width. Default height of block level elements is number of lines they have. We can change this behavior by giving custom height. Custom height and width not applicable to inline elements
  • Width can work in % and px in any case
  • Height can work in % if custom height of parent is specified
  • Height can work in px in any case
  • If we give custom height and width, child can cross the boundaries of parent
  • If we give custom height and width, contents mat flow out of the box
